# formatting

`formatting` is an Obsidian plugin that helps keep Markdown notes tidy while editing.

## Features

- Promote selected headings
- Demote selected headings
- Normalize blank lines automatically on save
- Preserve code fences and YAML frontmatter

## Spacing Rules

- Remove extra blank lines between normal paragraphs
- Remove blank lines around headings
- Keep exactly one blank line before and after blockquotes
- Keep exactly one blank line before and after horizontal rules
- Keep exactly one blank line before Markdown tables

## Development

- `npm run dev` to watch and rebuild during development
- `npm run build` to create a production bundle
